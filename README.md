# MASM-Casino

**Using multi-modules** <br>
* main file : 
  * FP.asm<br>
* game file : 
  * twentyOne.asm
  * ten.asm
  * bs2.asm
  * sm2.asm<br>
* include file ( view file ) : 
  * card.asm
  * backCard.asm
  * topBar.asm
  * statusBar.asm
  * lose.asm<br>
* library file : 
  * mylib.inc<br>
* batch file: 
  * make2.bat<br>

**A casino game including** <br>
* Black Jack 
* 11 & half 
* Slot machine 
* Big and small
